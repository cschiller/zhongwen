Options for the Zhongwen Chrome Extension

On this page you can set your personal preferences for the Zhongwen Chinese Pop-up Dictionary Chrome extension.

Save your changes here

The Tone colors setting allows you to turn colors for the pinyin syllables on and off. If tone colors are turned on, you will see colors corresponding to the tones of the characters: red = first tone, orange = second tone, green = third tone, blue = fourth tone.

The Tone Color Scheme setting allows you to choose which color scheme to use for the tone colors.

The Font Size setting allows you to change the size of the characters in the pop-up window.

The Simplified and Traditional Characters setting allows you to choose whether to display both simplified and traditional characters; or to automatically detect the character style used on the page and only display those characters in the pop-up window.

The Show Zhuyin setting allows you to turn on Zhuyin (Bopomofo) phonetic symbols.

The Pop-up color setting allows you to change the background color of the window showing the dictionary entries. (Changing this setting requires reloading the page with the pop-up to take effect.)

The Skritter domain setting allows you to change the URL that is used for adding words to your Skritter queue (Skritter login required). By default, skritter.com is being used, but for some users skritter.cn might work better.

The Show grammar and usage notes setting allows you to turn on grammar and usage notes.

Don't forget to save your changes by clicking on the botton at the top of this page.
