Zhongwen Help

Keyboard shortcuts

Navigation
N Next word
B Previous character ("back")
M Next character
Moving the pop-up window
A Move the pop-up location between three different locations
Y Move pop-up window down
X Move pop-up window up
Working with text
C Copy translation to clipboard
Ctrl + C Just copy the selected Chinese characters (without translation) to the clipboard
Learning words and characters
R Remember word by adding it to the internal word list
S Add word to Skritter queue
Alt + W Show the internal word list
External dictionaries
Alt + 1 LINE Dict
Alt + 2 Forvo
Alt + 3 Dict.cn
Alt + 4 iCIBA
Alt + 5 MDBG
Alt + 6 Reverso
Alt + 7 MoE Dict
T Tatoeba
